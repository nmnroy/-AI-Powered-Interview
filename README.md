## 🌟 Overview

The **AI-Powered Interview Prep Platform** is a cutting-edge web application designed to help job seekers ace their dream roles. By simply pasting a job description, our AI engines instantly generate highly relevant interview questions. You can practice your answers, receive real-time, comprehensive feedback from the AI, and track your proficiency across multiple sessions using our sleek Glassmorphism dashboard!

## ✨ Features

- **🤖 AI-Tailored Practice:** Generate targeted interview questions based strictly on the job roles/descriptions you want.
- **🎙️ Real-Time Feedback:** Get instant, actionable analysis and scoring on your answers via advanced LLMs.
- **📊 Glassmorphism Dashboard:** Track your "Skill Proficiency", perfect score trends, and session history through an interactive, premium dark-mode interface.
- **🌗 Seamless Theming:** Beautiful Light & Dark mode support out-of-the-box.
- **🔐 Secure Authentication:** Enterprise-grade security and user management powered by Clerk.

## 🛠️ Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) with Glassmorphism UI
- **Database:** [PostgreSQL via Supabase](https://supabase.com/) & [Prisma ORM](https://www.prisma.io/)
- **Authentication:** [Clerk](https://clerk.dev/)
- **AI Models:** Gemini & Groq Integration
- **Deployment:** [Vercel](https://vercel.com)

## 🚀 Quick Start (Local Development)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nmnroy/-AI-Powered-Interview.git
   cd -AI-Powered-Interview/clerk-nextjs
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the `clerk-nextjs` directory and add your keys (Clerk, Supabase, Gemini, Groq).

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open localhost:**
   Visit `http://localhost:3000` to view the app locally.
